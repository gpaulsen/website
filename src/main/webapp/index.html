Why not RTC?

There is no need to move away from RTC if you are happy with it. We decided to move to git and JIRA for following reasons:

  • performance
  • stability
  • offline capabilities
  • integration into various IDEs
  • integration into build tools
  • community support

Git

Learn how to migrate to git using rtc2git script or cli command by checking out our github pages for the cli exension or the script
